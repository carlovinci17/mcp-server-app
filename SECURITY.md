# Security Policy

This is a personal portfolio project (a fictional company, "Vortex Digital",
built to demonstrate Azure Functions' native MCP tool trigger extension). It
is not a maintained open-source project and has no formal vulnerability
disclosure program or bug bounty.

If you find a security issue - e.g. a way to access data or Azure resources
you shouldn't be able to, or a way to run up API costs against the deployed
demo - please open a GitHub issue or contact the repository owner directly
rather than exploiting it. There's no sensitive production data behind this
project (all customer/employee/document data is synthetic), but the Azure
resources themselves are real and billed.

## Scope

- The demo deployment at the domain linked in this repo's About section.
- The application code in this repository.

Infrastructure outside this repo (the Azure subscription, resource
configuration not captured in code) is out of scope for reports.
