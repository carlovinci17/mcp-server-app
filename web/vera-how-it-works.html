Retrieval strategies

Vera has three genuinely different ways to retrieve data — not one fixed pipeline. The agent decides which fits each question. Click a prompt to trace its actual path.

Exact lookup
Fetch by ID or a simple filter — plain SQL, no matching logic at all.
SQL substring search
SQL ILIKE '%query%' against specific columns — fuzzy, but not semantic.
Vector / hybrid RAG
Embeds the query, then BM25 + vector search together against Azure AI Search.
