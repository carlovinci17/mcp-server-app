Retrieval strategies
Vera has three genuinely different ways to retrieve data — the agent decides which fits each question. Click a prompt to watch its actual path.
Exact
Direct database lookup by ID or filter. Fast, deterministic, no matching logic.
SQL Search
SQL ILIKE text-pattern matching on specific columns. Fuzzy, not semantic.
Vector / Hybrid
Embeds the query, then BM25 + vector search together via Azure AI Search.
